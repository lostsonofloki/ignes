Ignes | Cinema Discovery, Refined by Fire.
Ignes is a privacy-first movie logging and discovery engine designed to replace generic algorithms with a personalized "Oracle." Built as a bespoke gift for a fellow cinephile and named after her daughter, Ember, the app focuses on the emotional and atmospheric core of film rather than just genre tags.

Key Features
Stoke the Embers (AI Discovery): A custom Gemini-powered engine that analyzes movie logs across all genres to find the common narrative threads in a user's taste.

The Inner Hearth: A deep-dive cinematic profile that identifies "Flashpoints"—the specific elements (tension, complexity, wonder) that drive a user's engagement.

Privacy-First Architecture: Opt-in AI discovery where user data is used only for real-time recommendations and never for model training.

High-Fidelity Tracking: A "StoryGraph-style" precision rating system (0.1 increments) and a 22-point mood palette.

Tech Stack
Frontend: React (Vite), Framer Motion, Recharts.

Backend: Supabase (PostgreSQL, Auth, RLS).

Intelligence: Google Gemini AI (LLM integration).

Data: TMDB & OMDb API.
